Task 2 - Data Visualization and Storytelling

Objective

Create visualizations that communicate business insights using sales data.

Tools Used

- Power BI
- Superstore Sales Dataset

Visualizations Created

1. Total Sales Card
2. Sales by Category
3. Sales by Region
4. Sales by Segment
5. Sales Trend Over Time

Key Insights

- Technology category recorded the highest sales.
- West region generated the most revenue.
- Consumer segment contributed the largest share of sales.
- Sales showed an overall upward trend over time.

Outcome

Learned data visualization, dashboard creation, and storytelling using Power BI.
